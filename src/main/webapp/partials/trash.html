Model Driven Engineering

Ingénierie dirigée par les modèles

Le développement d'application piloté par les modèles est un concept qui vise à se focaliser sur la modélisation de ce que doit faire l'application et de s'appuyer sur une génération de code automatique à partir de cette modélisation.
KOneEdit™ est un outil qui permet de modéliser tout ce que vous imaginez.
Extensible par méta-modélisation pour chaque modèle, il offre une infinité de possiblité de représenter les données et le comportement d'une application.

Rapid Application Development

Développement Accéléré d'Applications

Le développement d'une application peut-être long quand toute l'infrastructure technique est à faire. KNetThreadsPlatform™ accélère la création d'applications en proposant :
  • de modéliser le fonctionnel par des diagrammes UML à l'aide de KOneEdit™
  • de générer automatiquement le code permettant de compiler la modélisation en une application Java opérationnelle.
Cela permet à l'Architecte de l'application se concentrer sur l'organisation fonctionnelle, et au développeur de se concentrer sur l'implémentation métier, en se déchargeant de la technique bas niveau (gestion des threads, du réseau, etc.).

KNetThreadsPlatform™, c'est donc aussi :
  • un framework d'application multithread orienté 'messages'
  • une gestion unifiée des couches réseau via les Connector™ : un accès réseau est modélisé par un simpe 'module' dans le diagramme UML
  • un design soigné pour obtenir les meilleurs performances possible.

Note : Une multitude de Connector™ sont déjà disponibles (Http, Jdbc, Jms, Jsmpp, WebServices) et libres (OpenSource) ! Le framework Connector™ est extensible à tout type de communication réseau.

Web Ready

A la différence d'un simple site internet consultatif telle que la vitrine commerciale d'une société ou un blog, un Logiciel Web permet de mettre à disposition sur internet des calculs métier qui nécessitent une logique complexe, ainsi qu'une infrastructure serveur importante.
Néanmoins une part importante de cette infrastructure sera commune à beaucoup d'applications, comme la gestion des utilisateurs, de la sécurité... KWebPlatform™ implémente tout cela et est un formidable accélérateur de développement.
